# High Score
A program that calculates the highest score from a List of scores.

### Example Input
```
78 65 89 86 55 91 64 89
```
### Example Output
```
The highest score in the class is: 91.
```

## How to use
You can try the program acessing the [Replit](https://replit.com/@LukCnt/high-score?v=1) and clicking on the "Run" green button.
